# Bloom
An accountability tracker to help users reach personal goals, encourage self-improvement, and stay productive.

## Demo
https://kira-iovenko.github.io/accountability_tracker/

## Features
### Goals
- Create personal goals
- Customize each goal with an emoji and motivation
- Edit and/or delete goals
- Track daily completions
- Track longest streak
- Track streak automatically
- Monthly completion calendar
- Navigation through previous months

### Notes and Messages
- Add personal notes and messages for each goal
- Edit and/or delete functionality

### Journal
- Create journal entries
- Edit and/or delete entries
- Preview entries 

### UI
- Confetti animation upon goal completion
- Responsive card layout
- Dashboard view
- Goal detail overlay
- Clean pastel aesthetic

## Made using
- HTML
- CSS
- Vanilla JavaScript
- Local Storage API
